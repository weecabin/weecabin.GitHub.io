Infinite Flight Tools

Hold Patterns

Create flightplans that circle a user defined lat/lon with the specified number of loops and legs per loop.

Circling Approach

Create flightplans that dump you out on runway heading. if you select an airport runway as the destination, the pattern will start at the end of the downwind leg, and terminate at the runway threshold.

Radar Simulator

Practice vectoring any number of aircraft to a multiple runway airport.
After opening, click Start to run the simulator.
Add planes one at a time or continuously by respectively clicking "Add Plane" or "Add Planes".
"Add Planes" allows you to specify where planes will be added on the screen, and the delay between additions.
Specify an aircraft altitude or speed by clicking next to the aircraft to select it, then click either "Altitude" or "Speed" from the menu. Selected aircraft are painted green.
Planes with separation warnings are painted yellow, and alarms red.
If a selected plane has a separation issue, it will blink green then the warning or alarm color.
Use drag vector just like IF. The vector is sent when you lift your finger. When you put your finger down, the closest aircraft becomes the object of the drag vector.
Use two fingers to pan, and pinch to zoom in or out.
